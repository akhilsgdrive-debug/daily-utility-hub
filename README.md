# Daily Utility Hub

A clean, modern, lightweight collection of 10 everyday utility tools that run 100% in the browser.

**Live Demo:** https://akhilsgdrive-debug.github.io/daily-utility-hub/

## Features

- Calculator, Notes, To-Do, Countdown Timer, Stopwatch, Unit Converter, BMI, Age Calculator, Password Generator, Text Utilities

## Tech
Pure HTML + Tailwind CSS (CDN) + Vanilla JavaScript. Fully responsive, dark mode, localStorage persistence.

## Deployment
Designed for GitHub Pages. Push to `main` → Settings → Pages → Deploy from `main` branch.
